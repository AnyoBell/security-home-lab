# Security Home Lab

A virtualized environment for hands-on security practice.

## Overview
Home lab built to simulate real-world attack and defense scenarios.

## Stack
- **Hypervisor:** UTM
- **Attack:** Kali Linux
- **Defense:** Ubuntu Server
- **Monitoring:** Elastic Stack (SIEM)
- **Vulnerable Machine:** Metasploitable

## Structure
```
network/        # Network diagrams and topology
configs/        # VM and tool configurations  
detection/      # SIEM rules and alerts
writeups/       # Lab exercise documentation
```
